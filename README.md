# SvelteKit Template

A very simple SvelteKit template with the essentials.

# Content

* SvelteKit Skeleton template

* SCSS support
* Global SCSS file
* SCSS variable file

* EsLint support
* Follows [LVKSH rules](https://github.com/lvkdotsh/javascript)
* Prettier support
* Lint/Fix Command: `npm run lint` or `npm run fix`
